# Summarize a Proposal

**Category:** Sales / Proposals & Quotes
**Difficulty:** Beginner
**Time saved:** ~35 minutes per run

## What this does

Turns a long client proposal or RFP response into a one-page executive summary —
offer, price, timeline, risks, recommended next move.

## How to run

1. Open this workflow's page in the Operations Platform
2. Paste the proposal text (or upload the document)
3. Optionally enter the client name and pick the audience
4. Click **Launch**
5. The summary appears below; click "Copy to clipboard" or "Save to deal"

## Inputs

| Name | Required | Description |
|---|---|---|
| proposal_text | yes | The full proposal body |
| client_name | no | Used to personalize the heading |
| summary_audience | no | "Sales Leader" by default |

## Outputs

| Name | What it is |
|---|---|
| summary | One-page markdown summary |
| next_recommended_tasks | 2-4 suggested follow-ups |
